# 主流工具清单

下面按“一人公司高频任务”来分组：研究写作、图像内容、视频内容、代码生产力、知识管理与搜索。

## 对话与写作

- ChatGPT：通用对话、写作、数据分析与多模态；适合“一个入口解决多数问题”
- Claude：长文写作、总结与代码解释体验常被偏好；适合“文档密集型工作”
- Gemini：与 Google 生态协同使用；适合“多端与办公套件联动”
- Microsoft Copilot：Office/Windows 场景；适合“企业文档与会议协作”

## 搜索与研究

- Perplexity：以“搜索式问答”组织信息；适合“快速做资料综述”
- 传统搜索 + 摘要：Google/Bing + 摘要工具；适合“来源可追溯的调研”

## 图像与设计

- Midjourney：高质量风格化出图；适合“品牌视觉探索与海报”
- Stable Diffusion 生态（含 ComfyUI）：可控性强、可本地；适合“批量生成与工作流化”
- Adobe Firefly：与 Adobe 工作流结合；适合“商用素材管线”

## 视频与音频

- Runway / Pika / Luma 等：文生视频、图生视频、补帧与编辑；适合“短视频素材与广告片段”
- CapCut：剪辑工作流友好；适合“短视频生产”
- Descript：文本化剪辑、播客工作流；适合“内容创作者”
- ElevenLabs：配音与多语音；适合“旁白与多语言版本”

## 代码与开发生产力

- GitHub Copilot：IDE 内代码补全与建议；适合“加速日常编码”
- Cursor / 类似 AI IDE：更强的代码库理解与改动协作；适合“以改为主的开发”
- 通用 LLM + 代码工具链：适合“需要可控流程与可审计输出”

## 代理与应用搭建

- LangChain / LlamaIndex：应用编排、RAG、工具调用；适合“把能力产品化”
- AutoGen / CrewAI 等：多代理协作范式；适合“复杂任务拆解与协作”
- 低代码 Agent 平台：适合“更快出原型”

> 选型建议：如果你还没有稳定的“内容发布与获客”流程，优先把工具投入到“提速输出”而不是“造更复杂的系统”。
