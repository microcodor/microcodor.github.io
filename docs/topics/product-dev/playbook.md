# 从0到1路线图

## 1. 选题与验证（1-7 天）

- 明确人群：你为谁解决什么问题
- 找替代方案：他们现在用什么凑合
- 做承诺测试：写一个 landing + 收集邮件/预售

## 2. MVP（1-2 周）

- 只做一条主路径：用户完成核心任务的最短路径
- 只留一个付费点：不要同时做多种套餐与功能
- 交付优先：能人工交付的先人工，先卖再自动化

## 3. 上线与闭环（2-4 周）

- 观测：埋点/日志/告警完善
- 反馈：把用户问题整理成“需求队列”
- 留存：给用户一个持续回来的理由（更新/模板/数据/社区）

## 4. 增长与复利（持续）

- 内容：围绕你的关键词做系列文章/视频
- 渠道：SEO、社群、合作，先选一个打穿
- 产品化：把高频需求变成“按钮”，把交付变成“流程”
