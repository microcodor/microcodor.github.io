# 主流技术栈与工具

## 前端与全栈

- Next.js / Remix / Nuxt：适合做产品与营销站一体
- Vite + React/Vue：适合快速做前端应用
- Tailwind CSS：适合快速搭 UI

## 后端与数据

- Supabase / Firebase：快速获得鉴权、数据库、存储
- Postgres：产品化后的稳定数据库选择
- Redis：缓存、队列与限流

## 支付与订阅

- Stripe：全球化常用
- Paddle / LemonSqueezy：适合数字产品与税务处理需求

## 邮件与通知

- Resend / Mailgun / SendGrid：交易邮件与营销邮件
- Webhook + IM：把告警推送到飞书/Slack/企业微信

## 分析与观测

- PostHog / Plausible / GA4：行为分析与漏斗
- Sentry：前端/后端错误追踪
- UptimeRobot / Better Stack：可用性监控

## 交付与客服

- 文档：VitePress / Docusaurus
- 工单：Linear / Jira / Trello（按团队协作习惯）
- 社群：Discord / Telegram / 飞书群

> 选型建议：先选能让你“上线并收费”的最短路径，再逐步替换成更可控、更省成本的组件。
