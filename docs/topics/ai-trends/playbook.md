# 趋势转行动清单

## 1. 趋势筛选（10 分钟）

对每个趋势打分（1-5）：

- 和你用户的关系：是否直接影响他们的痛点
- 落地难度：一人是否能做出可交付版本
- 差异化：你是否有独特数据/经验/渠道

选总分最高的 1-2 个趋势进入下一步。

## 2. 产出一个“可验证物”

三选一即可：

- 一篇深度文章：用你的场景解释它的价值与边界
- 一个小工具/脚本：把趋势变成可演示的能力
- 一个工作流模板：让别人 30 分钟复现

## 3. 固化为长期资产

- 建评测集：同一任务用不同模型跑一遍，长期对比
- 建素材库：提示词、范式、模板、检查表
- 建渠道复用：SEO 词、视频脚本、newsletter 版式
