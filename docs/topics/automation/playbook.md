# 工作流模板与落地

## 1. 内容生产流水线（高 ROI）

- 触发：Notion/Airtable 新建选题
- 处理：生成大纲 -> 写初稿 -> 自检（事实/语气/结构）
- 输出：同步到博客仓库或 CMS，创建待发布任务
- 通知：发布后自动推送到邮件/社媒

## 2. 线索与客户支持自动化

- 触发：表单/邮箱/IM 收到新咨询
- 处理：分类（销售/支持/合作）-> 生成首轮回复 -> 建工单
- 输出：写入 CRM/表格，打标签与预计价值
- 通知：高价值线索单独提醒

## 3. 付费与交付自动化

- 触发：支付成功（Stripe/LemonSqueezy 等）
- 处理：创建用户/发放许可 -> 发送 onboarding 邮件 -> 建群邀请
- 输出：把发票、合同、交付物链接归档
- 通知：异常支付、退款、风控告警

## 落地检查表

- 失败策略：重试次数、失败告警、人工兜底入口
- 幂等性：同一事件重复触发不会造成重复扣费/重复发货
- 日志：每次执行至少记录输入、输出与关键分支
- 安全：密钥集中管理，最小权限原则
