# 常用体系与工具

## 任务与项目体系

- GTD：把“想法”变成“下一步行动”
- 周复盘：每周一次清空收件箱、重排优先级
- OKR/北极星指标：让你不被任务推着走

## 信息与知识管理

- PARA（Projects/Areas/Resources/Archives）：把资料按用途组织
- Zettelkasten：更适合写作者与研究者的笔记体系

## 时间与注意力

- 时间盒（Timeboxing）：给重要任务留出不可侵犯的时间
- 深度工作（Deep Work）：减少上下文切换
- 80/20：抓住最有效的少数动作

## 习惯与行为设计

- 原子习惯：用环境与反馈设计行为
- 先系统后目标：建立可重复的流程，而不是一次性冲刺

## 常见工具（按功能）

- 任务：Todoist / TickTick / Things
- 知识：Notion / Obsidian / Logseq
- 日程：Google Calendar / Apple Calendar
- 复盘与写作：任意 Markdown + 版本控制
