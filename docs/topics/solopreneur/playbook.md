# 拆解模板与复盘方法

## 一页拆解模板

复制下面结构，任何案例都能套进去：

- 目标用户：谁在为什么付费
- 核心痛点：他们最难的一步是什么
- 价值主张：你提供什么更好的替代方案
- 获客渠道：内容、社群、SEO、广告、合作，哪个是主力
- 转化路径：从看见到付费经历了哪些页面/触点
- 交付方式：SOP、模板、产品功能、服务流程
- 定价策略：按次/订阅/套餐/阶梯
- 复购机制：会员、持续更新、增值服务
- 关键指标：线索数、转化率、交付周期、留存、毛利

## 复盘输出物（建议最少三个）

- 行动清单：你能在 7 天内做的 3 个动作
- 资产清单：你需要沉淀的 3 个资产（模板/脚本/页面/流程）
- 风险清单：你最可能失败的 3 个点以及规避方式
